A lightweight, zero-dependency web app to generate UTM-tracked URLs for Google Analytics and marketing campaigns. Built with plain HTML, CSS, and Vanilla JS for instant use anywhere.

## Features
- **Real-time URL construction**: Enter base URL and UTM params (utm_source, utm_medium, utm_campaign, utm_term, utm_content) for instant previews.
- **Templates & Presets**: Save common setups via localStorage for reuse.
- **Bulk Builder**: Generate multiple links quickly.
- **Responsive Design**: Mobile-first, works everywhere.
- **Export Options**: Copy, QR code, or download as CSV.


## Tech Stack
- **Frontend**: HTML5, CSS3 (with Flexbox/Grid), Vanilla JavaScript (ES6+)
- **Deployment**: GitHub Pages, Netlify, or any static host
